# Agent Notes

Street Fight Dojo is an Arduboy project focused on special-move practice, fast feedback, and secrets.

## Standing Rules

- Create a `README.md` when a repo does not have one.
- Update the README as code changes.
- Arduboy first.
- Screen resolution is 128 x 64.
- Build the pocket dojo, not a fighting game or simulator.
- Move recognition first. Everything else second.
- Do not add features without approval.
- Secrets are gameplay and must be entered as controller inputs.

## Coding Standards

- Use 2-space indentation.
- Prioritize readability.
- Avoid unnecessary abstraction.
- Keep menus and loops small.
- Run `scripts\test-arduboy.ps1` before submitting changes.
