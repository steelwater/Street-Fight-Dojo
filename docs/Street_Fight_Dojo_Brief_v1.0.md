# Street Fight Dojo
## Crew Brief v1.0

Create a small Arduboy training game inspired by classic fighting game special move practice.

The player performs directional and button inputs. When a valid move is detected, the game displays the move name and plays a short sound.

Do not build a fighting game. Do not build a simulator. Build a pocket dojo.

## MVP

- Ryu, Ken, Guile
- Hadouken: Down, Down-Right, Right, A
- Shoryuken: Right, Down, Down-Right, A
- Tatsumaki: Down, Down-Left, Left, B
- Sonic Boom: charge Left for 1 second, then Right, A
- Flash Kick: charge Down for 1 second, then Up, B
- Visible input history
- Move name appears on success
- High score saves
- At least one secret code exists
